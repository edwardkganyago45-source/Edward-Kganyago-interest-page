# My Interests Website

## Project Description

My Interests is a personal website created using HTML and CSS. The website introduces who I am and showcases some of my favorite interests, including anime, football, and coding. The project was built as part of my learning journey in web development through CodeYourFuture and demonstrates my ability to create structured and styled web pages.

## Features

* Personal introduction section
* Information about my interests
* Ordered list of my top anime series
* Images related to my interests
* Custom styling using CSS
* Responsive and easy-to-read layout

## Installation Instructions

1. Download or clone the repository.
2. Open the project folder.
3. Locate the `index.html` file.
4. Open `index.html` in your web browser.

## Usage Instructions

Navigate through the webpage to learn more about my interests and background. The website serves as a personal profile and portfolio project.
